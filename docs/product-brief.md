# Product Brief

## Что это

Личный AI-инструмент для продвижения нескольких продуктов через статьи и автопостинг.

## Для кого

Для владельца продуктов, который не хочет вручную заниматься контент-маркетингом и не хочет ежедневно модерировать публикации.

## Что система должна делать сама

- придумывать темы;
- собирать контент-план;
- писать статьи;
- делать картинки;
- публиковать в блог;
- адаптировать под VK;
- публиковать по расписанию;
- повторять временно упавшие шаги.

## Когда пользователь вмешивается

- если что-то сломалось;
- если конкретный материал не понравился;
- если нужно изменить настройки продукта, аудитории или стиля.

## Главный KPI проекта

Минимум ручных действий при стабильном выпуске полезного контента по нескольким продуктам.
